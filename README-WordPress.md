# S&S Kreates Co. — WordPress HTML Blocks for Astra Theme

## Overview
WordPress-ready HTML blocks for the S&S Kreates Co. budgeting and savings challenge site.  
Design is minimal, pastel purple, butterfly and rose accents, accessibility first.

## File Structure

### HTML Pages
- `index.html` – Home with hero, categories, highlights, and call to action
- `budgeting-tools.html` – Digital budgeting tools and trackers
- `worksheets.html` – Worksheets and workbooks with category filters
- `scratch-cards.html` – Interactive scratch-off savings challenges

### CSS
- `additional-css.css` – Styles for WordPress → Customize → Additional CSS

## WordPress Block Structure

### Homepage Blocks
1. Header Navigation – Fixed bar with butterfly icon
2. Hero Section – Subtle animated gradient
3. Highlights – Why S&S Kreates Co.
4. Product Categories – By type and theme
5. Testimonials – Community stories
6. Call to Action – Join a challenge
7. Footer – Links and branding

### Product Page Blocks
1. Header Navigation
2. Page Header or Hero
3. Product Showcase Grid
4. Bundle Offer
5. Features or Comparison
6. Call to Action
7. Footer
