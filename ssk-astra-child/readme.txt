== Description ==
A pastel-forward Astra child theme for S&S Kreates Co with gamified savings layouts, WooCommerce-ready shop sections, and shortcodes for mini wins progress.

== Installation ==
1. Install and activate the Astra parent theme.
2. Upload the `ssk-astra-child` folder (or zip) via Appearance → Themes.
3. Activate "S&S Kreates Astra Child".
4. Create pages using the provided templates under Page Attributes.

== Shortcodes ==
[ssk_mini_wins_number] – Displays the current user's mini win count (shows 24 for guests).
[ssk_savings_progress_bar] – Renders a pastel progress bar for the user's mini wins (shows demo progress for guests).
